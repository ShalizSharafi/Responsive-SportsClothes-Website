## 📂 Features

- **Responsive Layout** – Works on mobile, tablet, and desktop.
- **Fixed Header** – Navigation stays accessible as you scroll.
- **Interactive Hover Effects** – Buttons and product images animate on hover.
- **Product Showcase** – Includes limited edition apparel and accessories.
- **Scrolling Highlight Bar** – Key features and values scroll horizontally.
- **Sections with Clear Purpose** – Product details, brand philosophy, and shop sections.

---

## 🖥 Sections Breakdown

### 1. Header
- Logo and site title
- Desktop navigation menu with hover underline effects
- Search and mobile menu icons
- Fixed at the top for accessibility

### 2. Hero / Collection Overview
- Large hero heading: *“for make”*
- Highlights collection details:
  - Collection: 15 items, apparel and essentials
  - Design: Everyday wear, sustainably made
  - Edition: Limited release, ends in 14 days
- “Shop All” call-to-action button

### 3. Scrolling Highlights
- Horizontal marquee highlighting features:
  - Essentials collection
  - Sustainable materials
  - Premium quality
  - Limited release
  - Worldwide shipping

### 4. Product Gallery
- 15 products displayed with hover animation
- Each product card shows:
  - Image (with hover zoom effect)
  - Product name
  - Price
- Examples:
  - Studio Polo – $200
  - Utility Crewneck – $150
  - System Tee – $80
  - Field Bottle – $40

### 5. Brand Philosophy
- Purpose: Support modern makers in daily work
- Approach: Minimal, functional designs
- Focus: Everyday essentials
- Philosophy: Less noise, more intention in design

### 6. Learn More Section
- Information about brand values and design approach
- CTA button: *“Learn More”*

### 7. Build Section
- Background hero image with text overlay
- Highlights how Wearvo supports modern makers
- CTA button: *“Shop All”*

## ⚡ Technologies Used

- **HTML5** – Semantic markup
- **Tailwind CSS** – Utility-first styling
- **CSS Animations** – Smooth hover effects and scrolling marquee
- **Responsive Design** – Optimized for all devices
